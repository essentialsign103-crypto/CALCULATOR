# TEHA_TECH Scientific Calculator

A responsive, browser-based scientific calculator supporting:

- Basic & scientific functions
- Trigonometry in DEG/RAD
- Memory store/recall
- History of calculations
- Light/Dark mode
- Keyboard input
- Mobile and PC friendly design

## How to Use

Open `index.html` in any modern browser.

## Features

- Responsive design for mobile and desktop
- History shows last 10 calculations
- Keyboard input for digits and operations
- Light/Dark toggle
